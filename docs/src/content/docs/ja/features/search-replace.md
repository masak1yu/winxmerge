## 検索

**Cmd+F** で検索バーを開きます。

- 検索フィールドの横に**マッチ数**を表示
- **前へ / 次へ** ナビゲーションボタン
- 2-way と 3-way の両方の差分ビューで使用可能

## 置換

検索バーには置換機能が含まれています：

- **Replace**: 現在のマッチを置換
- **Replace All**: アクティブなペインの全マッチを置換

## 行へジャンプ

**Cmd+G** で「行へジャンプ」ダイアログを開きます。行番号を入力するとその位置に直接ジャンプします。

## ブックマーク

- **Cmd+M**: 現在の差分行にブックマークを切替
- **F2**: 次のブックマークに移動
- **Navigate** メニューからブックマーク間を移動
