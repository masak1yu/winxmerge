WinXMerge は画像ファイルのピクセルレベル比較をサポートしています。

## 対応フォーマット

PNG, JPEG, GIF, BMP, WebP, TIFF, ICO

## 3パネルビュー

- **左パネル**: 元の画像
- **右パネル**: 変更後の画像
- **差分オーバーレイ**: 変更ピクセルを赤、同一ピクセルをグレーで表示

## コントロール

### ズーム

**10%〜400%** の連続ズームスライダーと、ウィンドウサイズに合わせてスケーリングする **Fit** モード。

### ブレンドスライダー

ブレンドスライダー（0〜100%）で、変更ピクセルのハイライトを左右パネルに調整可能な透明度で直接オーバーレイ表示。変更箇所をコンテキスト内で確認しやすくなります。

### 差分パネルの切替

差分オーバーレイパネルの表示/非表示を切り替えて、個々の画像に集中できます。
