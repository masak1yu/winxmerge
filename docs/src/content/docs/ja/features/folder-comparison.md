title: フォルダ比較
description: ディレクトリを再帰的に比較、フィルタリング・ソート機能付き。
---

WinXMerge は2つのディレクトリを再帰的に比較し、全ファイルのステータスを表示できます。

## 概要

ファイル選択ダイアログで2つのフォルダを選択するとフォルダ比較が始まります。表示内容：
- **ツリースタイルのインデント**（ネストしたディレクトリ）
- **ファイルステータス**: 同一 / 異なる / 左のみ / 右のみ
- **更新日時**（両側）
- **ファイルサイズ**（両側）

## フィルタ

### ステータスフィルタバー

リスト上部のステータスボタンでファイルリストをフィルタリング：
- **All**: 全ファイルを表示
- **Identical**: 同一のファイルのみ
- **Different**: 異なるファイルのみ
- **Left only**: 左ディレクトリにのみ存在するファイル
- **Right only**: 右ディレクトリにのみ存在するファイル

### .gitignore サポート

WinXMerge は `.gitignore` パターンを自動的に読み込み、マッチするファイルを除外します。`.git` ディレクトリは常に除外されます。

### ファイル拡張子フィルタ

特定のファイル拡張子で比較をフィルタリングできます。

## カラムソート

カラムヘッダーをクリックしてファイルリストをソート：
- Name（名前）
- Status（ステータス）
- Left Size / Right Size（左右のサイズ）
- Left Modified / Right Modified（左右の更新日時）

再度クリックで昇順/降順を切り替え（▲/▼ で表示）。

## 操作

- ファイルを**ダブルクリック**すると、新しいタブで詳細な差分ビューが開く
- **右クリック**でコンテキストメニュー：
  - 左にコピー / 右にコピー
  - 削除
- **< Back** ボタンでファイル差分からフォルダビューに戻る

## ZIP アーカイブ比較

`.zip` ファイルを仮想フォルダとして比較可能。CRCとファイルサイズに基づいて追加/削除/変更されたエントリを表示します。
