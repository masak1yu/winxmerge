title: エクスポート & レポート
description: 差分結果をHTML、Excel、CSV、JSONレポートとしてエクスポート。
---

## HTML レポート

**ファイル → Export HTML Report...** でスタイル付きHTMLファイルを生成：
- 色分けされた差分表示（アプリのビジュアルスタイルに一致）
- すべての差分コメントがレポートに埋め込み
- 共有や印刷に適したフォーマット

HTML レポートはシステムの印刷ダイアログからの印刷時にも使用されます。

## Excel レポート

**ファイル → Export Excel (.xlsx)...** で Excel ワークブックを生成：
- 色分けされた行（追加=緑 / 削除=赤 / 変更=黄）
- 差分メモを含むコメント列
- Excel、LibreOffice、Google Sheets で利用可能

## 全コメントのエクスポート

**ファイル → 全コメントをエクスポート (CSV/JSON)...** で全タブの差分コメントを収集してエクスポート：
- **CSV**: スプレッドシート互換フォーマット
- **JSON**: マシンリーダブルフォーマット

コメントの追跡や共有が必要なコードレビューワークフローに便利です。
