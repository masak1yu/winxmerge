## CSV / TSV 比較

WinXMerge は `.csv` / `.tsv` ファイルの比較に専用のテーブルビューを提供します。

### 機能

- **セルレベル差分**: 各セルを個別に比較し、変更セルをハイライト
- **区切り文字自動検出**: ファイルごとにカンマ（CSV）またはタブ（TSV）を独立して検出
- **引用フィールド対応**: 埋め込みの区切り文字や改行を含む引用フィールドに対応
- **カラムリサイズ**: カラムヘッダーの境界をドラッグしてリサイズ
- **区切り文字不一致警告**: .csv と .tsv を比較する際に警告を表示

### 編集

- **インラインセル編集**: セルをクリックして内容を編集
- **Undo/Redo**: Cmd+Z / Cmd+Shift+Z でセル編集操作の取り消し・やり直し
- **保存**: Cmd+S で CSV として保存。新規ドキュメントは Save As ダイアログが表示
- **再スキャン**: F5 で編集セルから差分を再計算

### 新規テーブルドキュメント

**ファイル → 新規 → Table** で空のテーブル比較を作成（初期グリッド 10x5）。

## Excel / スプレッドシート比較

WinXMerge は Excel やスプレッドシートファイルのテーブルビューでのセル差分比較に対応しています。

### 対応フォーマット

| フォーマット | 拡張子 |
|--------|-----------|
| Excel (最新) | `.xlsx` |
| Excel (旧形式) | `.xls` |
| Excel マクロ付き | `.xlsm` |
| OpenDocument | `.ods` |

### 機能

- 変更セルがハイライト表示
- マルチシートファイルの**シートセレクター** — シートごとに比較
- 読み取り専用の比較ビュー
