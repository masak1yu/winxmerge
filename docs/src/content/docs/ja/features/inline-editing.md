---
title: インライン編集
description: 比較ビューを離れずに差分ペインで直接テキストを編集。
---

WinXMerge は WinMerge スタイルのインライン編集をサポートしています — 差分ペイン内でテキストを直接編集できます。

## 仕組み

編集は常に**整列されたゴースト行ビュー**で行われます（別の編集モードはありません）。変更は差分表示に即座に反映されます。

## キー操作

| 操作 | キー |
|--------|-----|
| 新しい行を挿入 | Enter |
| 空行を削除 | Backspace（行頭で） |
| 行間を移動 | Arrow Up / Down |
| 差分を再スキャン | F5 |
| 元に戻す | Cmd+Z |
| やり直し | Cmd+Shift+Z |

## ゴースト行

一方のペインで新しい行を挿入すると、整列を維持するために他方のペインに**ゴースト行**が自動追加されます。

## 再スキャン

**F5** を押すと、現在の編集内容から差分を再計算します。複数の編集を行った後に更新された差分状態を確認するのに便利です。

:::caution
未保存の編集がある場合、自動再スキャンはデータ損失を防ぐためにファイル再読み込みをスキップします。手動の F5 再スキャンは常に現在のエディタ内容から再計算します。
:::

## 新規空白ドキュメント

ゼロから編集するための空の比較ドキュメントを作成：

- **ファイル → 新規 → Text**: 空の 2-way 比較
- **ファイル → 新規 → Table**: 空の CSV/TSV 比較（10x5 グリッド）
- **ファイル → 新規 (3-pane) → Text / Table**: 空の 3-way 比較
