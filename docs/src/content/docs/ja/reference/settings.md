---
title: 設定
description: WinXMerge の動作と外観の設定。
---

設定は **Edit → Options...** からアクセスし、`~/.config/winxmerge/settings.json` に保存されます。

## 外観

| 設定項目 | 説明 |
|---------|-------------|
| テーマ | ライト or ダーク |
| 言語 | English or 日本語 |
| フォントサイズ | エディタフォントサイズ（8〜32pt）、View → Zoom In/Out で調整可能 |
| 行の折り返し | ウィンドウ端での行折り返しの切替 |

## 比較

| 設定項目 | 説明 |
|---------|-------------|
| ホワイトスペースを無視 | 異なるホワイトスペースの行を同一とみなす |
| 大文字小文字を無視 | 大文字小文字を区別しない比較 |
| 空行を無視 | 差分で空行をスキップ |
| 改行コードを無視 | CR/LF の違いを無視 |
| 移動行の検出 | 移動行の検出とハイライト |
| シンタックスハイライト | tree-sitter ハイライトの有効/無効 |

## フィルタ

| 設定項目 | 説明 |
|---------|-------------|
| 行フィルタ | 比較から行を除外する正規表現パターン |
| 置換フィルタ | 比較前に適用する正規表現の検索・置換 |

詳細は[フィルタ](/ja/features/filters/)を参照してください。

## 自動再スキャン

| 設定項目 | 説明 |
|---------|-------------|
| 自動再スキャン | ファイル変更の自動検出（500msごとにポーリング） |

有効にすると、外部でファイルが変更された際に差分を再実行します。未保存のインライン編集がある場合は再読み込みをスキップします。

## 外部エディタ

| 設定項目 | 説明 |
|---------|-------------|
| エディタコマンド | 外部エディタでファイルを開くカスタムコマンド |

**ファイル → 外部エディタで開く** からアクセスできます。

## プラグイン

設定の詳細は[プラグインシステム](/ja/integrations/plugins/)を参照してください。

## セッションの永続化

WinXMerge はセッションをまたいで以下のタブごとの状態を保存・復元します：
- ファイルパスとエンコーディング
- 改行コードとタブ幅
- 差分のみモードとステータスフィルタ
- 差分コメント
- ウィンドウのズームレベル
