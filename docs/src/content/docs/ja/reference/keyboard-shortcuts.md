---
title: キーボードショートカット
description: WinXMerge のキーボードショートカット一覧。
---

## ファイル

| ショートカット | 操作 |
|----------|--------|
| Cmd+S | 左ファイルを保存 |
| Cmd+N | 新規比較 |
| Cmd+T | 新規タブ |
| Cmd+W | タブを閉じる |

## ナビゲーション

| ショートカット | 操作 |
|----------|--------|
| Alt+↓ | 次の差分 |
| Alt+↑ | 前の差分 |
| Alt+Home | 最初の差分 |
| Alt+End | 最後の差分 |
| F2 | 次のブックマーク |
| Cmd+G | 行へジャンプ |

## 編集

| ショートカット | 操作 |
|----------|--------|
| Cmd+Z | 元に戻す |
| Cmd+Shift+Z | やり直し |
| Enter | 新しい行を挿入（インライン編集） |
| Backspace | 空行を削除（行頭で） |

## 検索

| ショートカット | 操作 |
|----------|--------|
| Cmd+F | 検索と置換の切替 |
| Cmd+M | ブックマークの切替 |

## ビュー

| ショートカット | 操作 |
|----------|--------|
| F5 | ファイルの再スキャン |

:::note
Windows/Linux では **Cmd** を **Ctrl** に読み替えてください。
:::

アプリ内で **Help → Keyboard Shortcuts** からも完全なショートカット一覧を確認できます。
