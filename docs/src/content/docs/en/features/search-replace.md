---
title: Search & Replace
description: Find and replace text across diff panes.
---

## Search

Press **Cmd+F** to open the search bar.

- **Match count** displayed next to the search field
- **Previous / Next** navigation buttons
- Works in both 2-way and 3-way diff views

## Replace

The search bar includes replace functionality:

- **Replace**: Replace the current match
- **Replace All**: Replace all matches in the active pane

## Go to Line

Press **Cmd+G** to open the Go to Line dialog. Enter a line number to jump directly to that position.

## Bookmarks

- **Cmd+M**: Toggle a bookmark on the current diff line
- **F2**: Navigate to the next bookmark
- Navigate between bookmarks via the **Navigate** menu
