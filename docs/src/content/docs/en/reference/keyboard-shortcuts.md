---
title: Keyboard Shortcuts
description: Complete list of WinXMerge keyboard shortcuts.
---

## File

| Shortcut | Action |
|----------|--------|
| Cmd+S | Save left file |
| Cmd+N | New comparison |
| Cmd+T | New tab |
| Cmd+W | Close tab |

## Navigation

| Shortcut | Action |
|----------|--------|
| Alt+↓ | Next diff |
| Alt+↑ | Previous diff |
| Alt+Home | First diff |
| Alt+End | Last diff |
| F2 | Next bookmark |
| Cmd+G | Go to line |

## Editing

| Shortcut | Action |
|----------|--------|
| Cmd+Z | Undo |
| Cmd+Shift+Z | Redo |
| Enter | Insert new line (inline editing) |
| Backspace | Delete empty line (at start) |

## Search

| Shortcut | Action |
|----------|--------|
| Cmd+F | Toggle search & replace |
| Cmd+M | Toggle bookmark |

## View

| Shortcut | Action |
|----------|--------|
| F5 | Rescan files |

:::note
On Windows/Linux, replace **Cmd** with **Ctrl**.
:::

You can also view the complete shortcut list in-app via **Help → Keyboard Shortcuts**.
